## Up and running in 3 steps

**1. Install**

<Tabs>
  <TabItem label="Homebrew">
```sh
brew install straubt1/tap/tfx
```
  </TabItem>
  <TabItem label="Docker">
```sh
docker pull ghcr.io/straubt1/tfx:latest
```
  </TabItem>
  <TabItem label="Windows">
Download the latest binary from the [Releases page](https://github.com/straubt1/tfx/releases).
  </TabItem>
</Tabs>

**2. Configure**
```hcl
# ~/.tfx.hcl
tfeHostname     = "app.terraform.io"
tfeOrganization = "my-org"
tfeToken        = "my-token"
```

**3. Run**
```sh
tfx workspace list
```
